# 💼 Salary Excel Automation Project

## 📌 Project Overview

This project is built using Python and OpenPyXL to automate employee salary calculations in Excel.

## 🚀 Features

* Create Excel file automatically
* Add employee data
* Calculate Bonus (12%)
* Calculate Total Salary
* Apply styling to header
* Auto adjust column width

## 🛠 Technologies Used

* Python
* OpenPyXL

## 📊 Output

Excel file with:

* Employee details
* Bonus calculation
* Total salary
